Waarom gebruiken we hexadecimaal voor kleuren?

"Hex color values guarantee color fidelity and matching across a choice of 16,777,216 colors. HTML and CSS color choices are limited to 140 colors. 140 color names are defined in the HTML5 and CSS3 color specifications. 17 colors are from the HTML specifications, 123 colors are from the CSS specification."
